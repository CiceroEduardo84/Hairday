- `Javascript`
- `Babel`
- `webpack`

## Installation

```bash
# clone project
$ git clone https://github.com/CiceroEduardo84/Hairday.git

# install dependencies
$ npm install

# run simulation api
$ npm run server

# run interface
$ npm run dev
```
